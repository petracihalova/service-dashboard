# Favicons Folder

This folder contains favicon options for the Service Dashboard.

## Adding New Favicons

1. Add your favicon files to this folder
2. Supported formats: `.ico`, `.png`, `.svg`, `.jpg`, `.jpeg`
3. The favicon will automatically appear in the Favicon Manager
4. Access the Favicon Manager from the top navigation bar

## File Naming

- Use descriptive names (e.g., `red-hat-logo.png`, `team-icon.ico`)
- Avoid spaces in filenames (use hyphens or underscores instead)
- The filename (without extension) will be displayed in the Favicon Manager

## Recommended Specifications

- **Size**: 48x48 pixels or larger
- **Format**: `.ico` for best browser compatibility, `.png` for transparency
- **File size**: Keep under 100KB for optimal performance
