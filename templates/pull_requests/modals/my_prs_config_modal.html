My PRs Configuration

Environment Variables:

  • GITHUB_USERNAME - Your GitHub username for filtering GitHub PRs
  • GITLAB_USERNAME - Your GitLab username for filtering GitLab MRs

Examples:

export GITHUB_USERNAME="your-github-username"
export GITLAB_USERNAME="your-gitlab-username"

Usage:

When configured, the "My PRs" button will show only pull/merge requests created by your accounts:

  • GitHub PRs: Filtered by GITHUB_USERNAME
  • GitLab MRs: Filtered by GITLAB_USERNAME

Note: After changing environment variables, restart the application for changes to take effect.
