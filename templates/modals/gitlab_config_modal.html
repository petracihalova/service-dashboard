GitLab Username Configuration

Environment Variable: GITLAB_USERNAME

Description: Your personal GitLab username for filtering "My MRs"

Example:

GITLAB_USERNAME="your-gitlab-username"

Usage:

When configured, the "My MRs" button will show only merge requests created by your GitLab account.

Note: After changing the environment variable, restart the application for changes to take effect.
